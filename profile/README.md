## ğŸš€ Getting Started

### Quick Start with MCP Servers

#### PDF Reader MCP
```bash
# Via npx (easiest)
npx @sylphx/pdf-reader-mcp

# Via Docker
docker run -v /path/to/pdfs:/pdfs sylphx/pdf-reader-mcp
```

#### Filesystem MCP
```bash
# Via npx
npx @sylphx/filesystem-mcp

# Via Bun
bunx @sylphx/filesystem-mcp
```

#### RAG Server MCP
```bash
# Clone and run
git clone https://github.com/sylphxltd/rag-server-mcp
cd rag-server-mcp
npm install
npm start
```

### Integration with Claude Desktop
Add to your `claude_desktop_config.json`:
```json
{
  "mcpServers": {
    "pdf-reader": {
      "command": "npx",
      "args": ["-y", "@sylphx/pdf-reader-mcp"]
    },
    "filesystem": {
      "command": "npx",
      "args": ["-y", "@sylphx/filesystem-mcp"]
    }
  }
}
```
